M5 AI 备课模块

#### 一、模块目标
面向学生、教师提供 AI 智能备课能力，依托用户知识点、薄弱点、上传教材文档快速自动生成完整教学教案与配套 PPT 课件；支持自定义 PPT 样式、在线编辑修改、导出标准 PPTX 文件，生成的备课成果可一键同步至 M4 即时讲课模块，同时联动 M1 题库自动匹配配套练习题，形成「备课 - 授课 - 练习」完整教学闭环。
核心业务能力
1. 多渠道素材导入：手动文字输入、PDF/Word/PPT/ 图片上传、导入 M3 个人薄弱知识点；
2. AI 一键生成全套备课：教学大纲、课时重难点、课堂互动设计、例题、幻灯片结构；
3. PPT 定制化：系统模板选用、自定义配色字体、拖拽调整幻灯片顺序；
4. 备课内容全生命周期管理：草稿保存、编辑、归档、历史记录；
5. 文件导出与跨模块联动：导出 PPTX 课件、同步备课内容到讲课页面、绑定题库习题。

#### 二、整体实现方案
采用 Java+Python 双微服务架构分工开发，前端负责交互展示，Java 处理业务管理，Python 承载 AI 生成与文档解析。
1. 前端（React+TS）
页面能力：备课创建表单、文件上传、教案富文本编辑、PPT 幻灯片预览、模板样式配置页面；
交互逻辑：流式接收 AI 生成内容，实时展示大纲与幻灯片，提供下载、同步讲课等操作按钮。
2. Java Spring Boot 主后端
基础业务：统一接收前端请求，管理备课、PPT 模板数据；
存储能力：对接对象存储保存上传文件与导出课件；
跨模块互通：封装接口调用 M1/M3/M4，实现跨模块数据互通；
AI 中转：通过内部 HTTP 调用 Python AI 服务，使用 SSE 流式把生成内容透传给前端；
文件处理：基于 POI 完成 PPTX 文件导出；
运维控制：增加限流、权限、缓存控制。
3. Python FastAPI AI 微服务
文档处理：解析各类上传文档提取文本内容；
Prompt 组装：整合用户年级、薄弱点、教学风格拼接专属 Prompt；
AI 内容生成：调用大模型生成教案、PPT 分页结构；
流式输出：统一处理 AI 流式输出、上下文内容组装。
   
数据存储链路
MySQL：存储备课主体、PPT 模板数据；
Redis：缓存备课内容、模板信息、AI 调用限流；
MinIO/OSS：存放用户上传文件、导出 PPT 文件。

#### 三、模块所需配套资源
1. 数据库
共享复用表：`teaching_contents` 存储教案、PPT 结构；
新增数据表：`ppt_templates` 存放系统 / 用户自定义 PPT 模板配置。
2. 缓存 Redis
缓存备课详情、PPT 模板配置；
维护 AI 生成接口限流 key，限制用户高频调用。
3. 各端技术依赖
前端：文件上传组件、富文本编辑器、PPT 在线预览工具；
Java：POI（PPT 导出）、MinIO/OSS 存储 SDK、WebFlux 流式响应工具；
Python：PDF/Word/PPT 文档解析库、大模型调用 SDK、OCR 图片识别工具。
4. 外部依赖服务
第三方：大模型 API 接口、对象存储服务、图片 OCR 识别服务；
内部模块接口：M3 薄弱点查询、M4 讲课内容创建、M1 题库习题推荐接口。
   
#### 四、验收标准简述
1. 多类型素材均可正常生成完整教案与 PPT；
2. 支持编辑幻灯片、自定义模板配色，可正常导出可编辑 PPTX；
3. 备课内容能一键同步到即时讲课模块；
4. AI 生成过程流式展示，无卡顿、数据持久保存。
